From Equations to Vectors to Complex Numbers

Introduction

The idea of a matrix is so central to linear algebra that we often take it for granted. Most students know how to compute matrices, but when it comes to imagining what a matrix represents, many start to struggle.

So what exactly is a matrix? Some might say it’s a way to organize a system of equations. Others might think of it as a tool for representing multi-dimensional geometry. The truth is, a matrix can be all of these things and more. That’s what makes it so tricky to define — matrices are incredibly versatile, much like numbers in the way we use them.

In this post, I’ll walk through some of the ways we can interpret matrices. By the end, we’ll see that these interpretations are independent of one another and conclude that matrices are a structure in their own right, rather than a representation of a fixed concept.

Matrix as a System of Equations

One of the most natural ways to think about a matrix is as a tool for handling linear systems of equations.

Any linear system of equations can be rewritten in matrices, almost too conveniently. That is no coincidence because matrices were precisely invented for this purpose.

In the matrix form,

  • The first matrix contains the coefficients of the variables.
  • The second matrix (a column matrix) holds the unknown.
  • The right-hand side (also a column matrix) contains the constant terms.

Geometrically, solving a linear system is equivalent to finding the intersection of the graphs. In that case, each row of the coefficient matrix corresponds to the direction of the line. That is because each coefficient signifies how far you move in each direction per unit increase of a variable, much like “rise” and “run” in the two-dimensional case.

Rewriting a system of equations in terms of matrices brings powerful advantages. For example, one can also develop an algorithm to solve any system of equations, like Gaussian elimination.

Matrix as a Collection of Vectors

Matrices aren’t just for equations — they can also be seen as collections of vectors.

Instead of thinking row by row, let’s split a matrix into its columns. Each column itself is a vector, consisting of an array of coordinates. For example, the column matrix [3 2]ᵗ is the same as the vector <3,2>.

If you unpack the matrix multiplication, you will notice that the resulting product can be written as a combination of two column vectors. This means that the problem has shifted from finding the intersection of two lines to guessing a particular linear combination of two vectors.

A helpful way to think about this is:

“How many times in each direction (of column vectors) should I move to reach the target ?”

By drawing those vectors, we can confirm that the system of equations above indeed solves for the needed vector combination.

While matrices were originally designed to express equations, this vector perspective became an even more powerful tool in linear algebra. It allows us to think of a matrix as a machine that transforms and combines vectors in meaningful ways.

Matrix as Complex Numbers

So far, we’ve treated matrices as tools for equations and vectors. But there’s another perspective, in which a matrix behaves like a number itself.

Let’s take complex numbers as an example. A complex number consists of two parts: real and imaginary. It’s usually written as a + bi, where a is real and b is imaginary.

To reframe this structure in terms of matrices, we can start by representing every real number a as a 2×2 diagonal matrix:

This is almost trivial — it’s just repeating the number — but it lays the groundwork for extending to imaginary numbers. Also, notice that the number 1 corresponds to the identity matrix, which has the special property that multiplying any matrix by it leaves the matrix unchanged.

Next, we need a matrix to play the role of i, the imaginary unit defined by the property i² = −1. There is a 2×2 matrix whose square equals −1 times the identity matrix. With this construction, we no longer need a special symbol “i” — the real and imaginary parts are now distinguished by their positions within the matrix.

Combining the two parts, we can construct every complex number in terms of a 2×2 matrix.

This may look like a mere change of notation, but it has a nice consequence: it reinforces the idea that a complex number is a single number. The usual form, a + bi, misleadingly splits it into two parts. The matrix version keeps the two parts together and still behaves like a number under addition and multiplication.

Of course, not every matrix corresponds to a complex number. Only matrices of this special form do. That matches the fact that complex numbers have two degrees of freedom (a real and an imaginary part), while a general 2×2 matrix has four entries.

Conclusion

Matrices are powerful precisely because they are not tied to a single interpretation. We can view them as tools for solving systems of equations, as collections of vectors, or even as numbers in their own right. None of these perspectives cancels out the others; nor do they overlap with each other.

In that sense, a matrix is less a representation of one fixed idea and more a flexible framework. A matrix itself is just a rectangular array of numbers, and its meaning is derived from its interpretation. The application of matrices is not limited to the ones above, and that’s what makes matrices one of the most versatile concepts in practical mathematics.
